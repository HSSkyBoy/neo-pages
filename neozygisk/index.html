Zygisk Root / KernelSU

NeoZygisk-Fork

這是我基於原版 JingMatrix 的 NeoZygisk 專案所維護的強化分支。原專案透過 ptrace 實作 Zygote 注入，完美替代了 Magisk 內建的 Zygisk，並為 APatch 和 KernelSU 提供了強大的 Zygisk API 支援。

  • 防偵測強化：深度優化了模組在記憶體層級的注入邏輯，確保在面對設備虛擬化檢測（如針對 Root 環境的遊戲或銀行應用）時具備更強的隱匿能力。
  • 底層相容：繼承了對 KernelSU 最新介面的相容性，提供穩定且不易崩潰的執行環境。
