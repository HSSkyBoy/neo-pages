# NKBE.top - Personal Workspace 🚀

這是 [nkbe.top](https://nkbe.top) 的原始碼存放庫，作為我的個人入口導航頁與作品集展示中心。

![Screenshot](https://via.placeholder.com/800x400.png?text=Website+Screenshot+Here) 
## ✨ 特性 (Features)

- **現代化化 UI**：採用類似 ColorOS 水生設計與玻璃擬物化 (Glassmorphism) 風格。
- **純靜態架構**：不依賴繁重的框架，使用 Vanilla JavaScript 與 Tailwind CSS 構建，極致輕量。
- **資料驅動過濾器**：內建自定義的專案分類系統 (Android / Minecraft)，支援流暢的 DOM 元素進場動畫。
- **CDN 部署**：完美適配 Cloudflare Pages，實現 Git Push 即自動部署與全球加速。

## 🛠️ 技術棧 (Tech Stack)

- HTML5 / CSS3
- [Tailwind CSS](https://tailwindcss.com/) (透過 CDN 引入，免編譯)
- Vanilla JavaScript (Intersection Observer API)
- Lottie Web Player (微動畫)
