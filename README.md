## ✨ 特性 (Features)

- **現代化化 UI**：採用 Glassmorphism 風格。
- **純靜態架構**：使用 Vanilla JavaScript 與 Tailwind CSS 構建，極致輕量。
- **資料驅動過濾器**：內建自定義的專案分類，支援流暢的 DOM 元素進場動畫。

## 🛠️ 技術棧 (Tech Stack)

- HTML5 / CSS3
- [Tailwind CSS](https://tailwindcss.com/) (透過 CDN 引入，免編譯)
- Vanilla JavaScript (Intersection Observer API)
- Lottie Web Player (微動畫)
