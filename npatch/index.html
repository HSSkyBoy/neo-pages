NPatch

NPatch 是一個復刻自 LSPatch、以 LSPosed 為基礎的免 Root Xposed 框架，致力於為開發者與進階用戶提供穩定的模組注入環境。

  • 底層重構與優化：近期針對專案進行了深度的 Kotlin 語言遷移與 UI 介面重構，確保更現代化的程式碼結構與維護性。
  • 高強度簽名繞過：專注於簽名繞過邏輯與 Native 函式庫（確保 libnpatch.so 穩定 Patch 入目標應用程式）整合的底層解決方案。
  • 隱蔽性設計：透過精準的動態庫掛載機制，確保在無 Root 環境下的極致隱蔽性，減少應用程式的崩潰風險。
